# AStar_QuadTree_Visualization

## 📝 프로젝트 소개

이 프로젝트는 C++ DOS 콘솔 환경에서 알고리즘을 **‘보면서 이해할 수 있도록’** 만드는 시각화용 엔진입니다.

**마우스 기반 좌표 선택**, **A* 경로 탐색 시각화**, **쿼드 트리 분할 애니메이션** 등을 통해 길 찾기·공간 분할 알고리즘의 동작 과정을 직관적으로 확인할 수 있는 실험용 플레이그라운드를 목표로 설계했습니다.

## 💡 핵심 구현 내용

### 1️⃣ 사용자 경험(UX) 중심의 입력 시스템
*   **마우스 입력 지원**: 콘솔 환경임에도 불구하고 WinAPI를 활용하여 **직관적인 좌표 선택 UX**를 구현했습니다.
*   **인터랙티브 조작**: 키보드와 마우스를 조합하여 장애물을 설치하거나 목표 지점을 설정하는 등 능동적인 조작 환경을 제공합니다.

### 2️⃣ A* (A-Star) 알고리즘 시각화
*   **단계별 시각화**: 상태 머신(State Machine)을 기반으로 시작 지점 지정, 탐색 진행, 최종 경로 도출까지의 과정을 단계적으로 보여줍니다.
*   **속도 조절 시스템**: 경로 탐색이 진행되는 동안 **렌더링 속도를 실시간으로 조절**하여, 알고리즘의 탐색 흐름을 사용자가 원하는 속도로 관찰할 수 있도록 구현했습니다.

### 3️⃣ 쿼드 트리(Quad Tree) 분할 애니메이션
*   **공간 분할 시각화**: 노드에 객체가 추가될 때 4개의 하위 영역으로 나뉘는 과정을 시각적으로 표현합니다.
*   **분할 애니메이션**: 레벨 분할 요청 큐와 노드 타이머를 결합하여 십자 선이 그려지는 애니메이션을 구현함으로써, 데이터 구조의 변화를 직관적으로 이해할 수 있습니다.
